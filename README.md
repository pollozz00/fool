# Fool — Card Game
A client-server implementation of the classic card game "Durak" (Fool).

## Tech Stack
**Client**
- C++ / MFC / OpenGL
- GDI+ for texture loading

**Server**
- ASP.NET Core Web API (C#)
- Dapper + Microsoft SQL Server
- BCrypt password hashing

## Features
- Single-player vs AI (Easy / Medium / Hard difficulty)
- Player registration and login
- Rating system (Easy: ±10, Medium: ±25, Hard: ±50)
- Match history and leaderboard website
